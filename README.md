📝 To-Do List GUI Application (Python Tkinter)

A simple To-Do List desktop application built using Python and Tkinter.

This application allows users to add tasks, mark tasks as completed, and delete tasks using a simple graphical interface.
It is a beginner-friendly project that demonstrates the use of Python GUI programming, event handling, and list management.

🚀 Features





➕ Add Tasks – Users can enter a task and add it to the list.

✔ Mark Tasks as Completed – Selected tasks can be marked as completed with a checkmark (✓).

🗑 Delete Tasks – Remove tasks from the list easily.

🖥 Simple Graphical Interface built with Tkinter.

⚡ Lightweight and easy to use.






🛠 Technologies Used

Python

Tkinter (Python GUI library)

💻 How the Application Works

1. Enter a task in the input field.
  
2. Click ADD to add the task to the list.
 
3. Select a task and click MARK to mark it as completed.
 
4. Select a task and click DELETE to remove it.
